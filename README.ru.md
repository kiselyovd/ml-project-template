# ml-project-template

Продакшн-шаблон cookiecutter для ML-проектов: PyTorch Lightning / scikit-learn, Hydra, MLflow, DVC, FastAPI, Docker, CI/CD, публикация на HuggingFace Hub.

**English:** [README.md](README.md)

## Быстрый старт

```bash
pipx install cookiecutter
cookiecutter gh:kiselyovd/ml-project-template
cd my-new-project
make setup
make test
```
